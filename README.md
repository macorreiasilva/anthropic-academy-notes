# Anthropic Academy Notes

Repositório com meus estudos sobre Claude, API da Anthropic, prompt engineering, system prompts, conversas multi-turn e boas práticas no uso de LLMs.

## Conteúdos estudados

- Acesso à API da Anthropic
- Criação e proteção de API keys
- Primeiras chamadas com Python
- Conversas multi-turn
- System prompts
- Prompt engineering
- Segurança e boas práticas

## Estrutura do repositório

- `docs/`: resumos teóricos em Markdown
- `notebooks/`: experimentos em Jupyter Notebook
- `projects/`: miniprojetos práticos com Claude
- `prompts/`: coleção de prompts reutilizáveis
- `src/`: funções auxiliares em Python
- `exercises/`: questões e exercícios de revisão

## Segurança

Este repositório não contém API keys. Use um arquivo `.env` local baseado no `.env.example`.
